# 🧼 Handwashing Data Analysis (Dr. Semmelweis)

This project explores the groundbreaking discovery made by Dr. Ignaz Semmelweis on the importance of handwashing in reducing mortality rates.

Using data analysis and visualization, we examine how simple hygiene practices led to one of the most important breakthroughs in medical history.

---

## 📊 Project Overview

In the 19th century, Dr. Semmelweis discovered that handwashing drastically reduced deaths caused by childbed fever.

This project uses historical data to:

- Analyze death rates before and after handwashing was introduced
- Visualize trends in mortality
- Demonstrate the impact of data-driven decision making

---

## 🧠 Key Questions

- Did handwashing significantly reduce death rates?
- How did mortality trends change over time?
- What insights can we draw from the data?

---
